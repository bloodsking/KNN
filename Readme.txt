Cara memakai program kami

via google colab:
	1. download file .ipynb atau .py yang sudah kami sediakan di file
	2. buka "google colab" yang tersedia dan upload file .ipynb yang telah di download
	3. setelah terbuka, tekan tombol connect yang ada di bagian atas kanan
	4. jika ingin run secara keseluruhan bisa ke bagian runtime --> runall
	5. selesai program bisa dijalankan
